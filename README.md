# 鄉育教育系統：數位轉型與 AI 輔助學習案

> 人本教育 × 沉穩科技｜學習追蹤與洞察原型

## 📋 專案簡介

本專案為「鄉育教育基金會」的 AI 學習追蹤系統高保真原型，展示如何將碎片化的學習紀錄轉化為可視化的成長證據。系統以「人本教育」為核心，讓 AI 成為學生的陪伴式導航員，幫助學生把經驗說清楚、把能力命名、把下一步走得更踏實。

## ✨ 核心功能

### 1. 教務行政中心
- 智慧作業管理系統
- 30 秒快速點名功能
- 多角色切換登入介面（老師視角 / 學生視角）
- 課程進度管理與教材分發

### 2. AI 學習互動模組
- **導師視角**：AI 洞察實驗室
  - 學生作業深度分析
  - 思考深度摘要
  - 情感溫度計與預警系統
  - 技能點數變動預測
  - 誠實度與真實性偵測

- **學生視角**：成長導航
  - 智能導航與心得生成（4F 反思引導）
  - 我的成長歷程（時間軸視覺化）
  - 智慧數位履歷預覽（動態模板切換）
  - 自我覺察記事本
  - 學習助教通知中心

### 3. 成長數據儀表板
- **導師視角**：
  - 大數據驅動之游戲化「技能樹」
  - 團隊協作分析儀表板
  - 跨校能力對比

- **基金會管理者視角**：
  - 全台影響力儀表板
  - 跨校能力對比分析
  - 影響力證據鏈

## 🛠️ 技術棧

- **前端框架**：Vue 3 (CDN)
- **樣式框架**：Tailwind CSS (CDN)
- **圖表庫**：Chart.js (CDN)
- **圖標**：Emoji（已替換所有 Lucide Icons）
- **部署方式**：單一 HTML 文件，無需構建工具

## 🚀 快速開始

### 本地運行

1. 克隆或下載此專案
2. 直接用瀏覽器打開 `index.html` 即可

### GitHub Pages 部署

1. 將專案推送到 GitHub 倉庫
2. 在倉庫設置中啟用 GitHub Pages
3. 選擇 `main` 分支（或你的主分支）
4. 選擇 `/ (root)` 作為源目錄
5. 訪問 `https://[你的用戶名].github.io/[倉庫名]/` 即可查看
